To add a PDF or Word document: upload the file into the resources/files/ folder in the repo (via GitHub's "Add file → Upload files", or drag it into this folder in a Codespace), then add a link here, e.g. <a href="files/your-file.pdf">Your File</a>. There's no live upload button on the site itself — a static site with no backend can't accept uploads from visitors, so files always get added on the GitHub side first.
